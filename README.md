# EncryptSample

## 概要

EncryptSampleは、様々な暗号化アルゴリズムのサンプル実装を集めたプロジェクトです。このプロジェクトは、暗号理論の学習や実装の参考として、主要な公開鍵暗号方式をPythonで実装しています。各アルゴリズムは、クライアントとサーバーの通信を模したサンプルコードとともに提供されており、暗号化の基本的な動作を確認することができます。

## 含まれるアルゴリズム

### 1. ECDH (Elliptic Curve Diffie-Hellman)
楕円曲線を用いた鍵交換プロトコル。Diffie-Hellman鍵交換を楕円曲線上で行うことで、より効率的で安全な鍵共有を実現します。

### 2. ElGamal
公開鍵暗号方式の一種。離散対数問題の困難性に基づいています。暗号化とデジタル署名に使用可能です。

### 3. LWE (Learning With Errors)
格子ベースの暗号方式。量子耐性のある暗号として注目されています。Regevの公開鍵暗号方式を実装しています。

### 4. Pairing-IBE (Identity-Based Encryption using Pairings)
ペアリングを用いたIDベース暗号。ユーザーのID（メールアドレスなど）を直接公開鍵として使用できます。

### 5. RSA
古典的な公開鍵暗号方式。整数論の素因数分解の困難性に基づいています。

## プロジェクト構造

- `ECDH/`: ECDHの実装
- `ElGamal/`: ElGamalの実装
- `LWE/`: LWEと関連する格子暗号の実装
- `Pairing-IBE/`: IDベース暗号の実装
- `RSA/`: RSAの実装
- `*.txt`: 各アルゴリズムや数学の基礎についての説明文書

各アルゴリズムのフォルダには以下のファイルが含まれます：
- `client.py`: クライアント側のサンプルコード
- `server.py`: サーバー側のサンプルコード
- `*.txt`: アルゴリズムの説明
- `src/`: 共通のソースコード（曲線、フィールド演算など）

## 使い方

各アルゴリズムのフォルダに移動し、Pythonスクリプトを実行してください。例えば：

```bash
cd ECDH
python client.py
python server.py
```

各スクリプトは、暗号化の基本的な動作（鍵生成、暗号化、復号など）をデモンストレーションします。
