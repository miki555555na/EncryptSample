環：足し算、引き算、掛け算ができる集合
    ((a mod m) + (b mod m)) mod m = (a + b) mod m
    ((a mod m) - (b mod m)) mod m = (a - b) mod m
    ((a mod m) x (b mod m)) mod m = (a x b) mod m
    
    剰余環 Z/mZ：余りの世界{0,...,m-1}

    ※ 値を何乗したら何になるかは簡単に求められるが、何乗したらある値になるかを求めることは難しい(一方向性)
    → Diffie-Hellman鍵共有、Elgamal暗号、RSA暗号で安全性として採用

体：四則演算が閉じている集合
    整数の世界では割り算をすると結果が整数にならないことがあるが、有理数全体、実数全体、複素数全体では体になる。
    pが素数のときはZ/pZは、逆数を考えると割り算で閉じているので必ず体になる。(pが素数でないときは体になるとは限らない)
    → 素数pに対してZ/pZはFp、GF(p)とも表現される。有限個の元(集合の要素)からなる体なので有限体ともいう。

    有限体：任意のx ∈ Fp^* に対する逆数を考えると、pは素数で0 < x < pより、xとpは互いに素であり、gcd(x,p) = 1となる。
    　　　　拡張ユークリッド互助法を用いると ax + bp = gcd(x,p) = 1 となる整数a,bを見つけることができ、
    　　　　ax v 1 (mod p)となり、aはxの逆数になる。
    　
    　<巡回群の位数> <x> = {x^i|i=1,...,i-1}
    　　　Fp^* := Fp\{0} の乗法的巡回群の要素gに対して巡回群<g>の位数はp-1の約数になる。
    　　　gの位数aは定義より、g^a ≡ 1となる最小のa(a ≦ p-1)である。フェルマーの小定理よりg^{p-1} ≡ 1 (mod p)なので、aはp-1の約数になる。

    　<拡大体>
    　　　いくつかのFpをまとめたものに新しい四則演算を定義して、一つの体とと見なしたもの
    　　　Fp係数のn次多項式f(t)が既約なとき(これ以上分解できない多項式の時)、Fp[t]/(f(t)) は係数がFpで高々n-1次多項式の集まりになる。
    　　　この集合をFp^nと表し、Fpのn次拡大体という。拡大する前のFpを素体という。
    　　　例：F7上の既約多項式f(t)=t^2+1の解(i)を追加すると、a+biという形の数を作ることができ、2次拡大体F7^2を作れる。
    　　　
    　　　体kに対して、拡大体を作る操作を繰り返し、それ以上拡大体を作れないようになった体をkの代数的閉包という。
    　　　→ 「全ての多項式が解を持つ体」「多項式方程式が必ず解ける世界」になった。
    　　　→　実数の場合、Rにi = √−1 を追加するとCになり、複素数では全ての多項式が分解できる。CはRの代数的閉包である。(代数学の基本定理:すべての複素多項式は一次式に分解できる)
    　　　


DLP問題(離散対数問題)：g,p,g^a mod p が与えられたときにaを求める問題
DH問題：g,p,g^a mod p,g^b mod p が与えられたときにg^{ab} mod pを求める問題
　CDH仮定：DH問題が難しいという仮定
DDH問題：g,g^a,g^bが与えられているとき、g^{ab}またはランダムなcに対するg^cのどちらかが与えられたとき、どちらか判定する問題
※ DLPが解けるならCDHは解け、CDHが解けるならDDHが解ける -> DDH仮定が一番強い仮定(成り立つ確率が低い)

格子
    複素数集合Cの部分集合Aが次の二つを満たすとき格子であるという。
    　・λ1,λ2 ∈ Cが存在して、Λ　= Zλ1 + Zλ2と表せる。
    　・{λ1,λ2}はCのR上の基底である(つまり、C = Rλ1 + Rλ2)。
    格子は、加法+を介し、アーベル群の構造をもつ。

楕円曲線
    CをΛで割ったC/Λを複素トーラス TΛ と呼ぶ。C/Λの領域(平行四辺形)において、左端と右端、上端と下端を同一視でき、図形としてはドーナツ上の図形となる。
    複素平面上に同じ平行四辺形を隙間なく敷き詰めたとみなせる。
    敷き詰められた平行四辺形の角を格子点と呼ぶ。

    トーラス上の関数fを考えると、どの平行四辺形の上でも同じ値になる。
    z ∈ C,λ ∈ Λに対して、
    　f(z + λ) = f(z + Zλ1 + Zλ2) = f(z)
    となり、平行四辺形のなかのどこの点に対しても同じ値になる。
    複素数T ≠ 0と任意のzに対して以下の等式が成り立つとき、fは周期Tを持つという。
    　f(z + T) = f(z)
    すなわち、トーラス上の関数はλ1とλ2の2種類の周期をもつ(2重周期関数)。
    <ワイエルシュトラスのペー関数>　← 2重周期関数
    　　℘(z)= 1/z^2 + ∑ λ∈Λ\{0} (1/(z−λ)^2 - 1/λ^2)
    　微分方程式は、
    　　{℘′(x)}^2 = 4{℘(z)}^3 − 20α℘(x) - 28β (α,βは定数)
    　EΛ := {(x,y)|y^2 = 4x^3 - 20αx -28β} ∪ {∞} を考えると、
    　　 ϕ : TΛ ∋ z → (℘(z), ℘′(z)) ∈ EΛ
    　という写像を構成でき、1対1の関係であることから、トーラスTΛとEΛは同じものとみなすことができ、解の集合EΛを楕円曲線という。
 
    <ワイエルシュトラスの方程式>
    kを複素数体か標数pが2でも3でもない有限体とする。
    4a^3 + 27b^2 ≠ 0 を満たす a,b ∈ k を選ぶ。
    　y^2 = x^3 + ax + b 
    k上の楕円曲線 E/k を以下のように定義する。(¯kは代数的閉包、０は無限遠点)
    　E := { (x,y) ∈ {¯k}^2 | y^2 = x^3 + ax + b } ∪ {０}
    体h ⊆ ¯k を決め、点P = (x,y) ∈ Eについて、x,y ∈ hの時、点Pをh有理点、その集合E(h) を以下のように書く。
    　E(h) := { (x,y) ∈ h^2 | y^2 = x^3 + ax + b } ∪ {０} 

    足し算ができるため、ある点Pの整数倍も考えることができる。
     nP := P + P + ・・・＋P (n回足す)

楕円曲線離散対数問題
    楕円曲線上を一歩ずつ歩いた時、何歩歩いたか忘れてしまったとする。この時、現在地と自分の一歩から、何歩歩いたか知ることはできない
    すなわち、今いる現在地nPとPからnを知ることはできない。

    楕円曲線離散対数問題(ECDLP)：PとnPが与えられた時にnを求める問題
    →楕円曲線暗号はECDLPが難しいことを仮定している。

※ 複素平面上の領域Dで定義された関数f(z)がDの各点で微分可能ならばこの関数はDで正則であるという。
特異点：正則でない点(テイラー展開不可)

群
    足し算と掛け算のみに着目して抽象化した概念
    ある集合Gの2個の元(要素)に対してその集合の元を対応させる写像μを演算という。
    　μ：G x G → G, (x,y) → μ(x,y)
    集合G上の演算μ：G x G → Gが次の性質を満たす時Gはμに関して"群"であるという。μ(x,y) = x・y
        結合法則：任意のx,y,z ∈ Gに対して(x・y)・z = x・(y・z)
        単位元の存在：単位元と呼ばれるe ∈ Gが存在して任意の x ∈ Gに対してx・e = e・x ＝ x
        逆元の存在：任意のx ∈ Gに対してxの逆元x^{-1} ∈ Gが存在して、x ・x^{-1} = x^{-1}・x = e
    
    可換群(アーベル群)：上記に加えて以下の性質を満たす群
        任意のx,y ∈ Gに対して μ(x,y) = μ(y,x)
    
    位数|G|：群Gの元の個数。位数が有限の群を有限群、無限の群を無限群という。

    <群の例>
    ・実数全体の集合R：足し算に関して群(無限群)。掛け算に関しては0の逆元が存在しないため群にはならない。(0を取り除いた集合R^*は掛け算に関して群になる)
    ・有限体Fp：足し算に関して群(有限群)。掛け算に関しては0の逆元が存在しないため群にはならない。(0を取り除いた集合Fp^*は掛け算に関して群になる)
    ・楕円曲線上の点の集合：足し算に関して群。単位元は原点0。

    べき乗
        群の結合法則より任意のx ∈ Gに対して x・x・x を考えた時、結果は演算の順序によらず同じになる。
        一般にn個のxの演算は以下のように書け、xのn乗と呼ぶ。
        　x^n := x・x・...・x (n-1回演算)
        ※ 楕円曲線上の演算は足し算であったため、nPと表現する。

    巡回群
        群Gの元gを一つとり、<g> := {g^i|i∈Z}とすると、この集合は元の群の演算に関して群になる。
            結合法則：Gで成立しているため、部分集合<g>でも成立する。
            単位元の存在：g^0=eが存在する。
            逆元の存在：任意のg^iに対してその逆元g^{-i}も<g>の元である。
        g^i・g^j = g^{i+j} = g^j・g^i より、<g>は可換群である。
        Gが有限群のとき、部分群である<g>も有限集合であり、<g>は同じ値を持つことはわかる。
        → 途中で単位元に戻ってグルグル回っている。
        <g>を"巡回群"、gを<g>の"生成元"という。

    抽象レイヤー：巡回群<g>
    ・「ある要素 g を何回も演算（足す・掛ける）すると、いつか元に戻る」というルール。
    ・このルールがあるから、g^n（n回操作した結果）という計算が定義できる。

    具体レイヤー
    ・有限体の乗法群:：「大きな整数の掛け算 mod p」の世界。
    ・楕円曲線：「曲線上の点同士の足し算」の世界。
    
        
ペアリング
    楕円曲線E上の2個の点の組からある有限体Fqへの写像
    　e：E x E → Fq
    楕円曲線上の2点P,Qが与えられると原点O,P,Q,R:=P+Qで作られた平行四辺形OPQRが決まる。
    この平行四辺形の面積S(P,Q)とし、QがOを中心にPの反時計回りの方向にある時を正、逆向きにある時を負とする。
    面積S(P,Q)を、有限体のある固定値gの指数に乗せたものがペアリングの値になる。
    　e：E x E → Fq, (P,Q) → g^{S(P,Q)}
    P=Qのとき、S(P,Q)=0になるのでペアリングの値 e(P,P) = 1となる。
    
    一般に、整数a,bに対して以下の等式が成り立つ。
    　e(aP,bQ) = e(P,Q)^{ab}

    双線形性：E上の2点の和のペアリングはそれぞれのペアリングの積になる性質
        　P1 = a1e1 + b1e2, P2 = a2e1 + b2e2 のとき、P1 + P2 = (a1 + a2)e1 +(b1 + b2)e2より、
        　　e(P1 + P2, Q) = g^{(a1+a2)d−(b1+b2)c} 
    　　　　　　　　　　　　　= g^{(a1d−b1c)+(a2d−b2c)}
    　　　　　　　　　　　　　= e(P1, Q)e(P2, Q)
    　　　同様に、
    　　　　e(P, Q1 + Q2) = e(P,Q1)e(P,Q2)

    対称ペアリング・非対称ペアリング
        PとQが異なる楕円曲線上の点からなるペアリングが構成されることもある。
        ある楕円曲線には、楕円曲線のある点Pをその巡回群<P>に含まれない別の点Qに移す線型写像ψが存在する。
        　e'：E1 x E2 → Fq
        効率的に計算できる同型写像(ψ(ab)=ψ(a)ψ(b)かつ全単射) ψ1：E1 → E2, ψ2：E2 → E1の存在有無によって3タイプに分けられる。
        1. E1 = E2 
        2. E1 ≠ E2 かつ ψ2が存在する(及びψ1の存在は知られていない)。逆向きの写像が簡単に計算できない。←トラップドアとして暗号設計に利用できる
        3. E1 ≠ E2 かつ ψ1及びψ2の存在が知られていない。E1とE2の構造は似ているものの、具体的な変換式が見つけられない。←攻撃者がE1の情報をE2に持ち込んで悪さできないため、鍵サイズを効率的に設計できる。

        タイプ1   ：対称ペアリング
        タイプ2,3 ：非対称ペアリング
        ※対称ペアリングは用意するパラメータが少なく、ペアリングの順序を気にしなくて良かったが、対称ペアリングでよく使われていた曲線に対するDKPを解く方法が進展したため、非対称ペアリングが使われるようになった。

    MOVリダクション：ECDLPをDLPに帰着できる考えを利用した攻撃。当初、ペアリングは楕円曲線暗号を攻撃するための道具であった。
        　　　　　　言い換えると、DLPが困難なら、ECDLPも困難である。
        　　　　　　aPとbPが与えられた時DH問題はペアリングを使用することで解けるようになった。
    
    CBDH問題：楕円曲線上の点Pと整数 a,b,cに対して (P,aP,bP,cP) が与えられたときにe(P, P)^{abc}を求めよ。
    　CBDH仮定：CBDH問題が難しいという仮定。

    Weilペアリング e : E[m] × E[m] → ¯k  (¯kは代数的閉包)  
        E[m] := { P ∈ E | mP = O } であり、λ1,λ2 ∈ Cで定まるトーラスを考えると、E[m] = { (i/m)λ1 + (j/m)λ2 | 0 ≤ i, j < m } で点Pはm^2個ある。
        一般に、体の標数pがmを割り切らないとき、m^2個の点は¯kのどこかに散らばっている。
        　E[m]　⊂　 E(¯k) ≃ (Z/mZ)^2
        ある次数kの拡大体k^dを取ると、その中にすべてのE[m]が収まる。この最小のdを埋め込み次数と呼び、ペアリング暗号の安全性を決める重要な指標となる。
        Weilペアリングe_m(P, Q)の計算結果は、¯kの中の要素。正確には「1のm乗根」の集合 μm = { ζ ⊂ ¯k | ζ^m = 1}になる。

        〜イメージ〜
        　方程式を満たす「すべての可能性(核大体の点)」を考えた舞台(E(¯k))の中から選ばれたm^2個の点(E[m])から二つ選び、¯kの中にある「1のm乗根」へと対応させる道具。
        　楕円曲線上の点の情報を、この「拡大体の中に浮かんでいる1のm乗根」に写像している

        e(aP,bQ) = e(P,Q)^{ab}
        　ペアリングは、楕円曲線上の点の足し算を有限体上の掛け算へと変換している！
        　MillerのアルゴリズムはWeilペアリングを計算するための"計算機"の役割を果たす。

        <Millerのアルゴリズム>
        　Weilペアリングの定義に出てくるfpは、因子div(fP) = m(P) - m(０) を満たす。 すなわち、fPはP以外に零点を持たず、０以外に極を持たない関数。
        　← 楕円曲線上の点Pがm次ねじれ点(mP=０)である時に限り、このようなfPが存在する
        　このfPを求めるためには通常とてつもない時間がかかるため、Millerのアルゴリズムが使われる。
        　
        　中間関数f{i,P}を定義する。
        　div(f{i,P}) = i(P) -(iP) - (i-1)(０)
        　i=mのとき、 div(f{m,P}) = m(P) -(mP) - (m-1)(０) = m(P) - m(０) となる。

        　f{i+j,P}を、f{i, P}とf{j, P}から作るステップ
        　　楕円曲線上の点 U = iP と V = jP を足して W = (i+j)P を作る。
        　　1. L{U,V}: 点 U と V を通る直線。U, V, -W の3点で曲線と交わる。
        　　2. V{W}: 点 W を通る垂直線。W と -W で曲線と交わる。
        　Millerの更新式
        　　f{i+j, P} = f{i, P}・f{j, P}・L{U,V} / V{W}

        　Millerのアルゴリズムで求める「4つのパーツ」
        　 定義式 e(P, Q) = { fP(Q+S) / fP(S) } ・ { fQ(-S) / fQ(P-S) }を計算するために、Millerのアルゴリズム Miller(点, 代入先) を以下のように呼び出す。
        　　　1. fP(Q+S): Miller(P, Q+S) を実行
        　　　2. fP(S): Miller(P, S) を実行
        　　　3. fQ(-S): Miller(Q, -S) を実行　
        　　　4. fQ(P-S): Miller(Q, P-S) を実行
        　　これら4つの数値（すべて拡大体の要素）が揃えば、あとはそれらを掛け算・割り算するだけで Weilペアリングの値が求まる。
